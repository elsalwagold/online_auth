Online Odoo Authentication

Seamless Authentication and Attendance Synchronization

Key Features

Authentication Form: A website form that lets you authenticate to your online Odoo instance with preset credentials.

Attendance Synchronization: Sync attendance records from your local instance to a remote Odoo instance.

Employee Mapping: Map local employees to remote employees using employee numbers.

Sync Reports: Track synchronization history and results.

API Endpoints: Access attendance data through REST API endpoints.

Authentication Process

The module provides a simple web form to authenticate to your remote Odoo instance. The form is pre-populated with default credentials, but you can customize them as needed.

Once authenticated, you can access the remote instance's data and synchronize attendance records between the two instances.

Attendance Synchronization

The module allows you to synchronize attendance records between your local and remote Odoo instances. You can preview the records before synchronization and track the synchronization history.

The synchronization process maps employees between the two instances using employee numbers, ensuring that attendance records are correctly associated with the right employees.

API Access

The module provides an API endpoint to access attendance records:

  • GET /online/auth/attendance: Returns all attendance records in JSON format

This allows you to integrate the attendance data with other systems or applications.
